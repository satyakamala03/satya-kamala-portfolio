# Public Assets

This directory contains static assets that will be served at the root URL.

## Files to Add

1. **resume.pdf** - Your resume PDF file
   - Place it directly in this directory
   - The download button in the Hero section will link to it

2. **images/profile.jpg** or **images/profile.png** - Your profile picture
   - Place it in the `images/` subdirectory
   - See `images/README.md` for details

## Favicon

You can also add a custom favicon:
- `favicon.ico` - For the browser tab icon
